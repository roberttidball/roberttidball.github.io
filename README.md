This is a clean, modern, and fully static website built with:

* **HTML5:** For semantic structure.
* **CSS3:** For all styling, using CSS Variables for an easy-to-theme dark mode.
* **Vanilla JavaScript:** For smooth scrolling, scroll-based animations, and sticky header logic.
* **Google Fonts:** Using the 'Inter' typeface for a clean, professional look.

No frameworks or libraries were used, ensuring the site is extremely lightweight and fast.
